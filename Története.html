Git

A Gitet Linus Torvalds hozta létre, a Linux kernel fejlesztője, 2005-ben. Azért hozta létre, mert elégedetlen volt a meglévő verziókezelő rendszerekkel, amelyek lassúak és bonyolultak voltak. A célja az volt, hogy egyszerűen kezelhető, gyors és hatékony rendszert hozzon létre, amely lehetővé teszi a nagy projektek verziókezelését is. A Git alapelve az elosztott verziókezelés. Ez azt jelenti, hogy a fejlesztőknek saját lokális tárolóval rendelkeznek, amelyben a teljes projekt verziótörténete megtalálható. Ez lehetővé teszi a párhuzamos fejlesztést és a könnyű ágkezelést. A Gitet gyorsan elfogadták a fejlesztői közösségek, és ma a legelterjedtebb verziókezelő rendszerré vált.

GitHub

A GitHub 2008-ban alapították Chris Wanstrath, PJ Hyett és Tom Preston-Werner. Az alapítók célja egy könnyen használható, távoli Git tárhely szolgáltatás létrehozása volt. A GitHub megkönnyíti a projektek tárolását, nyomon követését és együttműködését. Elősegíti a közösségi fejlesztést és lehetővé teszi a felhasználóknak, hogy hozzájáruljanak más projektekhez.

A GitHub rendkívül népszerűvé vált a nyílt forráskódú projektek között. Ingyenes tárhelyszolgáltatást kínál nyílt forráskódú projektek számára, és számos funkciót kínál, mint például a problémák nyomon követése, a pull kérések, a projekt táblák és a hozzájárulások áttekintése. A GitHub a fejlesztők közösségi központjává vált, ahol az emberek megoszthatják, felfedezhetik és hozzájárulhatnak projektekhez.

Azóta a GitHub hatalmas felhasználói közösséget alakított ki, és fontos szerepet játszik az open-source projektek, vállalatok és fejlesztői csoportok közötti együttműködésben. A Microsoft 2018-ban felvásárolta a GitHubot, de a platform továbbra is függetlenül működik és népszerűsége folyamatosan növekszik.

A logón szereplő macska

A GitHub logóján található macskafej a híres "Octocat", ami a GitHub eredeti maskottja és ikonikus szimbóluma, habár ő inkább egy polip-macska, mint egy átlagos cicamica. Az Octocat először 2007-ben jelent meg és azóta szerves részévé vált a GitHub arculatának. Az Octocat egy fiktív karakter, amely egy macska és egy polip kombinációja. A neve a nyolc karra (octo) és a macskára (cat) utal, jelképezve a többoldalúságot és a kreativitást a fejlesztői projektekben.

Összesítés

A Git és a GitHub története közvetlenül kapcsolódik egymáshoz. A Gitet Linus Torvalds hozta létre, hogy hatékony és egyszerű verziókezelést biztosítson a fejlesztők számára. A GitHub a Git alapú tárhelyszolgáltatás, amely 2008-ban alakult, és lehetővé teszi a projekt-tárolást és az együttműködést. A GitHub a fejlesztői közösségek és a nyílt forráskódú projektek kiemelkedő platformjává vált. Ma a Git és a GitHub közpös a fejlesztők számára, akik hatékonyan kezelhetik a verziókezelést és együttműködhetnek projektekben.
